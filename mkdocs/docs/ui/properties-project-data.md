# Properties — Project Data

The **Project Data** panel lives in the **Properties editor** (the panel with the scene icon, usually the second icon from the top in the Properties sidebar). It is the central location for defining all the named lists that MaStro uses across the project.

All lists defined here are shared by every object in the scene.

---

## Mass

The **Mass** section groups the lists that apply to Mass and Block objects.

### Block

A list of named urban blocks. A block is a named grouping of buildings — for example, *Block A*, *Block 01*, or any label meaningful to your project.

Each entry has a name. A colour swatch is shown in the list for quick visual identification. Use the **+** button to add a new block, and the arrow buttons to reorder them.

### Building

A list of named buildings. A building is a sub-grouping within a block — for example, *Tower 1*, *Podium*, *Annex*.

---

## Architecture

The **Architecture** section groups wall and floor type definitions.

### Wall

A list of wall types. Each wall type has:

| Property | Description |
|---|---|
| **Name** | A label for the wall type |
| **Thickness** | Wall thickness in metres |
| **Offset** | Distance the wall is shifted from its centreline |
| **Colour** | Overlay colour shown in the 3D viewport |

### Floor

A list of floor types. Each floor type has a name. Floor types can be assigned to faces in Edit Mode via the Architecture panel.

---

## Street

A list of street types. Each street type has:

| Property | Description |
|---|---|
| **Name** | A label for the street type |
| **Width** | Total road width in metres |
| **Radius** | Corner radius at intersections, in metres |
| **Colour** | Overlay colour shown in the 3D viewport |
