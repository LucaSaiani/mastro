# Properties — Street

The **Street** sub-panel of Project Data defines the street types used across the project.

A list of street types. Each street type has:

| Property | Description |
|---|---|
| **Name** | A label for the street type |
| **Width** | Total road width in metres |
| **Radius** | Corner radius at intersections, in metres |
| **Colour** | Overlay colour shown in the 3D viewport |

Street types are assigned to edges in Edit Mode via the [Street](sidebar-street.md) sidebar panel.
