# Properties — Architecture

The **Architecture** sub-panel of Project Data defines wall and floor types used across the project.

---

## Wall

A list of wall types. Each wall type has:

| Property | Description |
|---|---|
| **Name** | A label for the wall type |
| **Thickness** | Wall thickness in metres |
| **Offset** | Distance the wall is shifted from its centreline |
| **Colour** | Overlay colour shown in the 3D viewport |

Wall types are assigned to edges in Edit Mode via the [Architecture](sidebar-architecture.md) sidebar panel.

---

## Floor

A list of floor types. Each floor type has a name. Floor types are assigned to faces in Edit Mode via the [Architecture](sidebar-architecture.md) sidebar panel.
