# Properties — Block

A list of named urban blocks — groupings of buildings (e.g. *Block A*, *Block 01*).

Each entry has a name and a colour swatch for quick visual identification. Use **+** to add, **×** to remove, and **↑ / ↓** to reorder entries.

Blocks are assigned to Mass and Block objects via the [Mass](sidebar-mass.md) sidebar panel.
