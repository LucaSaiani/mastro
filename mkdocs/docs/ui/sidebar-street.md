# Street Panel

The **Street** panel appears in the MaStro sidebar tab when the active object is a **Street** object. It is only active in **Edit Mode** with **Edge Select** mode enabled.

Select one or more edges and use the **Street Type** dropdown to assign a type from the project list (defined in the Properties editor). The street type carries the width and corner radius that Geometry Nodes uses to generate the road surface.

!!! tip
    Street geometry is generated per-edge. You can mix different street types in the same Street object — for example, a main road branching into residential streets — by selecting the relevant edges and assigning the appropriate type.
