# Street Panel

The **Street** panel appears in the MaStro sidebar tab when the active object is a **Street** object.

## Edit Mode

The panel is only active in **Edit Mode** with **Edge Select** mode enabled. Select one or more edges to assign their street type.

**Street Type**  
A dropdown listing all street types defined in the project (via the Properties editor). Selecting a type assigns it to all selected edges. The street type carries the width and corner radius that Geometry Nodes uses to generate the road surface.

---

!!! tip
    Street geometry is generated per-edge. You can mix different street types in the same Street object — for example, a main road (wider) branching into residential streets (narrower) — by selecting the relevant edges and assigning the appropriate type.
