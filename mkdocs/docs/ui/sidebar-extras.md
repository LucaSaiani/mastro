# Extras Panel

The **Extras** panel appears in the MaStro sidebar tab for any MaStro object in Edit Mode. It is **collapsed by default**.

The panel provides a single custom floating-point attribute per element type. These attributes have no fixed meaning — they are available as free-form user data that can be read by custom Geometry Nodes or used for any purpose that requires a per-element float value.

## Vertex Select Mode

**Custom Vertex Value**  
A float attribute stored per vertex (`mastro_custom_vert`). Applies to all selected vertices.

## Edge Select Mode

**Custom Edge Value**  
A float attribute stored per edge (`mastro_custom_edge`). Applies to all selected edges.

## Face Select Mode

**Custom Face Value**  
A float attribute stored per face (`mastro_custom_face`). Applies to all selected faces.
