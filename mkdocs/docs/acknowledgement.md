# Acknowledgement

## MaStroGIS

The GIS functionality in MaStro is based on two open-source Blender addons: [BlenderGIS](https://github.com/domlysz/BlenderGIS) by Domlysz, which provides basemap tile fetching and geographic scene management, and [blosm](https://github.com/vvoovv/blosm) by Vladimir Elistratov, which handles OpenStreetMap data import and Google 3D Tiles. Both are released under the GNU General Public License v3.
