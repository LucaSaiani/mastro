Read Slope

The Read Slope node analyzes the inclination of a surface and outputs its slope as numeric values and visual indicators. It is designed for architectural and technical workflows, allowing slopes to be read either as ratios or as angular values.

Inputs

Geometry

Input geometry used to evaluate the slope.

Representation Type

Defines how the slope is measured: either as a ratio or in degrees.

Triangulate

Triangulates the input geometry to improve measurement accuracy.

Show Geometry

Displays the input geometry together with the slope annotations.

Text

Displays the numeric slope values.

Font Size

Size of the slope text.

Decimals

Number of decimal places displayed in the slope value.

Offset from Geometry

Distance between the numeric value, the arrow, and the surface.

Offset from Arrow

Distance between the numeric value and the arrow.

Text Color

Color of the slope value text.

Arrow

Displays an arrow indicating the direction of the slope.

Length

Length of the slope arrow.

Radius

Radius of the arrow shaft.

Vertices

Resolution of the arrow geometry.

Tip Scale

Scale of the arrow tip.

Stem Factor

Controls the proportion between the arrow shaft and its tip.

Arrow Color

Color of the slope arrow.

Outputs

Geometry

Output geometry containing the slope annotations. This includes arrows and numeric values, and optionally the original input geometry.

Arrows

Geometry representing the slope arrows.

Text

Geometry representing the numeric slope values.

Slope Ratio

Computed slope value expressed as a ratio.

Slope Degrees

Computed slope value expressed in degrees.
