Dimension

The Dimension node generates linear dimensions from the input geometry. Dimension values are calculated based on the length of the referenced geometry edges and are intended for architectural and technical annotations.

Inputs

Geometry

Input geometry used to generate the dimensions.

Units

Units used for the dimension values: meters, decimeters, centimeters, or millimeters.

Show Units

Displays a unit suffix next to the dimension value.

Font Size

Size of the dimension text.

Decimals

Number of decimal places displayed in the dimension value.

Offset

Distance between the dimension text and the dimension line.

Font Color

Material applied to the dimension text.

Dimension Line

Symbol Size

Size of the dimension symbols.

Line Offset

Distance between the dimension line and the referenced geometry edge.

Extension Line

Toggles the visibility of the extension lines.

Dimension Color

Color of the dimension line.

Outputs

Geometry

The generated dimension geometry. Dimension lines are created as Grease Pencil objects, while the numeric values are generated as mesh objects.
